ARRAY

    In C language, arrays are reffered to as structured data types. An array is defined as finite ordered collection of homogenous data, stored in contiguous memory locations.
    Here the words,

  • finite means data range must be defined.
  • ordered means data must be stored in continuous memory addresses.
  • homogenous means data must be of similar data type.

Application:

  • To store list of Employee or Student names.
  • To store marks of a students.
  • To store list of numbers or characters etc.

To declare an array

Snytax :
      data-type variable-name[size];

Example:
       int arr[10];

Explanation:
       int is the data type, arr is the name of the array and 10 is the size of array. It means array arr can only contain 10 elements of int type. Index of an array starts from 0 to size-1 i.e first element of arr array will be stored at arr[0] address and last element will occupy arr[9].

To create an array

Syntax:
       data_type array-name[size] = { list of values };

Example:
       int marks[4]={ 67, 87, 56, 77 }; //integer array initialization
      float area[5]={ 23.4, 6.8, 5.5 }; //float array initialization

Example:

    #include<stdio.h>
    #include<conio.h>>
    void main()
    {
        int arr[4];
        int i, j;
        printf("Enter array element");
        for(i=0;i<4;i++)
        {
            scanf("%d",&arr[i]); //Run time array initialization
        }
        for(j=0;j<4;j++)
        {
            printf("%d\n",arr[j]);
        }
    }

Two dimentional array

C language supports multidimensional arrays. The simplest form of the multidimensional array is the two-dimensional array.

Syntax :
       data_type array-name[row-size][column-size]

Example:
       int a[3][4];

Example:
     #include<stdio.h>
    #include<conio.h>
    void main()
    {
        int arr[3][4];
        int i,j,k;

        for(i=0;i<3;i++)
        {

        }
    }
