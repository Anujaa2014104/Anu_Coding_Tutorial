DATA TYPES

                                 A programming language is made to help programmer to process certain kinds of data and to provide useful output .A program may need different types of data types to used to solve a perticular problem.



Integer Data Type:

This data type can hold integer values ie the whole numbers . It occupies 2 byte of memory. The keyword used to declare is int.

Syntax : int variable_name;

Example :        int a=10;       int num=25;

Float Data Type:

This data type can hold floating point values ie the decimal numbers with 6 digits of precision. It occupies 4 byte of memory.The keyword used to declare is float .

Syntax : float variable_name;

Example :        float a=10.234;       float num=2.5;

Character Data Type:

This data type can hold the single character . It occupies 1 byte of memory.The keyword used to declare is char .

Syntax : char variable_name

Example :        char ch='a';       char num='p';

Void Data Type:

This type do not have any specific values.We can use it for both 'int' and 'float'.It is normaly used to specific the type of the function.
