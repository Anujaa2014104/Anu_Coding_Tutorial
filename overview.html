Overview of C Language
  1. C is a general purpose language.

  2. It was developed in AT & T Bell laboratories of USA in 1972, designed and written by DENNIS RITCHIE.

  3. It is a structured programming language.

  4. It supports functions that enable the easy of code by breaking maintainability large code into simpler modules.

  5. It is considered as a high level language because it allows the programmer to concentrate the problem at hand and not worry about the machine that the program will be using.

Different parts of C program
  1. Pre-processor
  2. Header file
  3. Function
  4. Variables
  5. Expression
  6. Comment

1.Pre-processor

   #include, the first word of any C program. It is also known as pre-processor. The main work of pre-processor is to initialize the environment of program.

2.Header file

   Header file is a collection of built-in functions that help us in our program. Header files contain definitions of functions and variables which can be incorporated into any C program by pre-processor #include statement. Standard header files are provided with each compiler, and cover a range of areas like string handling, mathematical functions, data conversion, printing and reading of variables. To use any of the standard functions, the appropriate header file must be included. This is done at the beginning of the C source file. For example, to use the printf() function in a program, the line #include<stdio.h>is responsible.

3.main() function

   main() function is a function that must be used in every C program. A function is a sequence of statement required to perform a specific task. main() function starts the execution of C program.
