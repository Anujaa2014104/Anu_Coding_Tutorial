VARIABLES

    A variable is a name that may be used to store a data. Unlike constant, variables are changeable, it can change value of a variable during execution of a program. A programmer can choose a meaningful variable name.

Example : average, height, age, total etc.


Rules to define variable name:

    1.It must be upto 8 characters.
    2.It must not start with a digit and special character.
    3.It can consist of alphabets, digits and special symbols like underscore _.
    4.Blank or spaces are not allowed.
    5.Keywords are not allowed as variable name.


Declaration of variable:

    Declaration of variables must be done before they are used in the program. Declaration does two things.
    1.It tells the compiler what the variable name is.
    2.It specifies what type of data the variable will hold.
